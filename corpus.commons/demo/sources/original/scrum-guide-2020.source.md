**Provenance.** Schwaber and Sutherland first co-presented Scrum at the OOPSLA Conference in 1995. The first formal *Scrum Guide* was written in 2010 and has been periodically revised; the November 2020 edition is the current canonical version as of 2026. The Guide is the definitive primary source for the Scrum framework and is referenced by Letaw (2024) as the source for Scrum-related framework definitions in the *Handbook of Software Engineering Methods*.

**Role in corpus.** Retro-axis canonical primary source. The Scrum Guide is the definitive document defining the Sprint Retrospective as a named ceremony (alongside the other Scrum events). It is the primary source for any retro-axis work in this corpus, analogous to how TC 25-20 is the primary source for the AAR axis. The demo currently ships projections onto `decision-making` and `stakeholder-engagement` only; a dedicated retro axis is a candidate for a later corpus move.
